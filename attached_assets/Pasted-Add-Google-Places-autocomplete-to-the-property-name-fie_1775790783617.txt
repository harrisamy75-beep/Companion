Add Google Places autocomplete to the property name field on the Stays page.

name, brand, and location when they select one.

Step 1 — add GOOGLE_PLACES_API_KEY to Replit Secrets.
The user will need to get a key from console.cloud.google.com,
enable "Places API" and "Maps JavaScript API".

Step 2 — new backend route artifacts/api-server/src/routes/places.ts:

GET /places/search?q=searchterm
Proxies to Google Places Text Search API so we never expose 
the API key to the frontend:

const response = await fetch(
  `https://maps.googleapis.com/maps/api/place/textsearch/json` +
  `?query=${encodeURIComponent(q)}&type=lodging` +
  `&key=${process.env.GOOGLE_PLACES_API_KEY}`
);
const data = await response.json();
