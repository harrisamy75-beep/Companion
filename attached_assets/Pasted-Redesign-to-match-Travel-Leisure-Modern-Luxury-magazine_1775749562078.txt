Redesign to match Travel + Leisure / Modern Luxury magazine aesthetic —
warm, photographic, bold typography, feels like a premium lifestyle brand
not a tech product.

Step 1 — Typography:
Install these Google Fonts:
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:ital,wght@0,400;0,700;1,400;1,700&family=Raleway:wght@300;400;500;600&display=swap');

- Playfair Display: all headings and display text
- Raleway: all body, nav, labels, UI text

h1, h2, h3, h4 { font-family: 'Playfair Display', serif; }
body, p, input, button, label, span, a { font-family: 'Raleway', sans-serif; }
