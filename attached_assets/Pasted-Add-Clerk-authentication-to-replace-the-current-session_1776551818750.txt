Step 9 — Update ALL fetch calls in the frontend:
Clerk handles auth via its own session tokens, not cookies.
Add the Clerk token to every API request:

import { useAuth } from "@clerk/clerk-react";
const { getToken } = useAuth();

// In each fetch call:
const token = await getToken();
fetch("/api/travelers", {
  headers: {
    "Authorization": `Bearer ${token}`,
    "Content-Type": "application/json"
  }
})

Create a shared helper in src/lib/api.ts:
export async function apiFetch(
  path: string, 
  options: RequestInit = {},
  getToken: () => Promise<string | null>
) {
  const token = await getToken();
  return fetch(path, {
    ...options,
    headers: {
      "Content-Type": "application/json",
      ...(token ? { Authorization: `Bearer ${token}` } : {}),
      ...options.headers,
    },
  });
}

Use apiFetch everywhere instead of raw fetch.
